# Public folder images → Cloudinary URLs

Images from [`public/`](./public/) uploaded to Cloudinary folder `rajasthan-pipe-traders/public`.

To refresh after adding files under `public/`, run:

```bash
cd backend
npm run upload:public
```

That updates [`backend/data/public-cloudinary-urls.json`](./backend/data/public-cloudinary-urls.json) and [`backend/data/public-cloudinary-urls.md`](./backend/data/public-cloudinary-urls.md).
